# 卡通 2D 图标提示词（统一风格）

目标：2D 卡通、圆润、粗描边、柔和配色、无复杂背景、透明底 PNG（建议 512×512）。

通用风格提示词（中文）：
- “2D 卡通风格天气图标，圆润可爱，粗描边，柔和粉蓝配色，轻微阴影，高对比轮廓，干净透明背景，居中构图，PNG，512x512”

通用风格提示词（英文）：
- “2D cartoon weather icon, cute rounded shapes, thick outline, soft pastel palette, subtle shadow, high-contrast silhouette, centered, transparent background, PNG, 512x512”

建议生成集合：
- `clear.png`：晴天太阳
- `cloudy.png`：多云
- `rain.png`：下雨（云 + 雨滴）
- `snow.png`：下雪（云 + 雪花）
- `thunder.png`：雷雨（云 + 闪电）

落地规则：
- 保持同一画风、线宽、阴影方向
- 透明背景（不要白底）
- 图标边缘留 6–10% 安全边距，避免任务栏小尺寸裁切
